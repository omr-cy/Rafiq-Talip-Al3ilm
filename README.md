# رفيق طالب العلم (TalibAlIlm Companion)

تطبيق متكامل مصمم خصيصاً لخدمة طلاب العلم الشرعي، يهدف إلى تنظيم الرحلة العلمية وتوثيق الفوائد والمجالس بطريقة عصرية وأنيقة تعتمد فلسفة "الفخامة الهادئة" (Quiet Luxury).

## 🌟 أهداف التطبيق
- **تنظيم الطلب:** توفير بيئة مركزة لتنظيم المتون، الدروس، والمواعيد العلمية.
- **حفظ الفوائد:** توثيق المجالس العلمية والتدبرات والتحقيقات بطريقة يسهل الرجوع إليها.
- **تثبيت العلم:** استخدام نظام البطاقات التعليمية (Flashcards) لتعزيز الحفظ والمراجعة.
- **الخصوصية التامة:** الحفاظ على بيانات المستخدم محلياً بالكامل دون الحاجة لخوادم خارجية.

## ✨ المميزات الرئيسية

### 1. نظام البطاقات التعليمية (Flashcards)
- إنشاء بطاقات تعليمية مقسمة حسب التصنيفات (فقه، عقيدة، لغة، إلخ).
- دعم نظام التكرار المتباعد لتثبيت الحفظ.
- واجهة تفاعلية أنيقة تدعم النصوص المنسقة.
- أداء عالٍ يدعم آلاف البطاقات بفضل نظام التحميل التدريجي (Pagination).

### 2. معجم المصطلحات الشخصي (Dictionary)
- بناء معجم خاص لطالب العلم لتدوين المصطلحات الغريبة والتعريفات الدقيقة.
- نظام بحث سريع وتصنيف للمصطلحات حسب الفنون العلمية.

### 3. المجالس والتدوين (Notes)
- محرر نصوص متطور يدعم التنسيق الكامل.
- قوالب جاهزة (مجلس علمي، تخريج حديث، تدبر آية).
- إمكانية إضافة وسوم (Labels) لتنظيم الملحوظات.

### 4. الأهداف والعادات (Goals & Habits)
- تتبع الأهداف العلمية (قراءة كتب، حفظ متون).
- نظام تتبع العادات اليومية (ورد القراءة، الأذكار، إلخ).
- ربط المهام بالعادات لضمان الاستمرارية.

### 5. صندوق الوارد (Inbox)
- ميزة فريدة لكتابة رسائل لنفسك المستقبلية (أدعية، نصائح، أهداف بعيدة).
- قفل الرسائل وفتحها تلقائياً في تاريخ محدد مع تنبيه المستخدم.

### 6. التسجيلات الصوتية (Audio)
- تسجيل المجالس العلمية والدروس مباشرة داخل التطبيق.
- إمكانية إضافة تفريغ نصي (Transcription) لكل تسجيل.

### 7. آية اليوم (Daily Ayah)
- عرض آية مختارة يومياً مع تفسيرها (المختصر في التفسير) لربط طالب العلم بالقرآن الكريم.

### 8. نظام التنبيهات الذكي
- إشعارات أنيقة وغير مزعجة لبداية المهام والمواعيد النهائية.
- تنبيهات لمراجعة البطاقات وفتح رسائل الوارد.

## 🛠 الجوانب التقنية
- **الخصوصية:** جميع البيانات تحفظ محلياً باستخدام `IndexedDB`.
- **النسخ الاحتياطي:** دعم كامل لتصدير واستيراد البيانات بصيغة `JSON`.
- **التصميم:** واجهة مستخدم متجاوبة (Responsive) تدعم الوضع الليلي وألواناً مريحة للعين.
- **الأداء:** استخدام تقنيات التحميل التدريجي والتمرير اللانهائي لضمان السرعة مع البيانات الضخمة.

## 📱 التوافق
التطبيق مصمم ليعمل كـ تطبيق ويب متطور (PWA) وكـ تطبيق أصلي على أجهزة أندرويد باستخدام Capacitor.
